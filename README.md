# hello-kafka - Hello World Kafka example

# REQUIREMENTS

* [docker-kafka](https://github.com/wurstmeister/kafka-docker)
* [Java](http://www.oracle.com/technetwork/java/javase/downloads/index.html) 1.8+
* [Gradle](http://gradle.org/) 2.7+

# EXAMPLE

```
$ cd kafka-docker/
$ docker-compose up
...

$ cd hello-kafka/
$ gradle cucumber
...
```
